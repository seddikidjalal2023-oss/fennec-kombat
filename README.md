# 🦊 فينك كومبات (Fennec Kombat)

لعبة Tap-to-Earn على تيليجرام، بشخصية الفنك مرتديًا زيًا جزائريًا تقليديًا (جندورة بيضاء بتطريز أخضر، وشاشية حمراء).

هذا الدليل يشرح لك **كل شيء من الصفر**، حتى لو لم يسبق لك إنشاء بوت أو استضافة مشروع من قبل.

---

## الخطوة 1: الحصول على توكن البوت (Bot Token)

1. افتح تيليجرام وابحث عن الحساب الرسمي: **@BotFather**
2. أرسل له الأمر: `/newbot`
3. سيطلب منك اسمًا للبوت (مثال: `Fennec Kombat`)
4. سيطلب منك اسم مستخدم فريد ينتهي بـ `bot` (مثال: `FennecKombat_bot`)
5. سيرسل لك رسالة تحتوي على **التوكن** — سلسلة طويلة تشبه:
   `7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
6. احتفظ به سريًا تمامًا — هذا التوكن هو "كلمة سر" بوتك.

---

## الخطوة 2: رفع المشروع على GitHub (مطلوب لمعظم خدمات الاستضافة المجانية)

1. أنشئ حسابًا على **github.com** إن لم يكن لديك واحد
2. أنشئ مستودعًا (Repository) جديدًا باسم `fennec-kombat`
3. ارفع مجلد المشروع بالكامل إليه (يمكنك استخدام واجهة الرفع في الموقع مباشرة عبر "Add file" ← "Upload files" إن لم تكن معتادًا على Git)

---

## الخطوة 3: الاستضافة المجانية عبر Railway

اخترت **Railway** لأنه الأسهل للمبتدئين ولديه خطة مجانية تكفي لبوت صغير.

1. ادخل إلى **railway.app** وسجّل عبر حساب GitHub
2. اضغط **New Project** ← **Deploy from GitHub repo**
3. اختر مستودع `fennec-kombat` الذي رفعته
4. من إعدادات المشروع (Variables)، أضف المتغيرات التالية:
   - `BOT_TOKEN` = التوكن الذي حصلت عليه من BotFather
   - `WEBAPP_URL` = سيتم تعبئته في الخطوة التالية بعد معرفة الرابط
5. بعد أول نشر (Deploy)، سيعطيك Railway رابطًا عامًا مثل:
   `https://fennec-kombat-production.up.railway.app`
6. انسخ هذا الرابط وضعه في متغير `WEBAPP_URL` ثم أعد النشر (Redeploy)

---

## الخطوة 4: ربط الرابط بالبوت كـ Mini App

1. عد إلى **@BotFather**
2. أرسل `/mybots` واختر بوتك
3. اضغط **Bot Settings** ← **Menu Button** ← **Configure Menu Button**
4. أرسل رابط تطبيقك (نفس رابط Railway) ثم اسمًا للزر مثل "🎮 العب الآن"

من الآن، أي شخص يفتح محادثة مع بوتك سيرى زر تشغيل اللعبة مباشرة.

---

## الخطوة 5: التشغيل محليًا للتجربة (اختياري، على حاسوبك)

```bash
npm install
cp .env.example .env
# افتح .env وضع فيه BOT_TOKEN الحقيقي
npm start
```

ثم افتح المتصفح على `http://localhost:3000` لتجربة واجهة اللعبة مباشرة (بدون تيليجرام، تعمل بحساب تجريبي محلي).

## هيكل المشروع

```
fennec-kombat/
├── server.js          # الخادم + منطق البوت + واجهة برمجية (API)
├── db.js              # قاعدة بيانات SQLite لحفظ تقدم اللاعبين
├── package.json
├── .env.example        # مثال المتغيرات السرية
└── public/
    ├── index.html      # واجهة اللعبة
    ├── style.css        # التصميم (ألوان جزائرية: أخضر/أحمر/أبيض + كثبان الليل)
    ├── game.js          # منطق النقر والطاقة والمهام اليومية من جهة المتصفح
    └── assets/
        ├── fennec.svg          # شخصية الفنك المرسومة
        └── project-thumb.svg   # أيقونة المشروع
```

## الميزات المُنفّذة فعليًا (وليست مجرد واجهة)

- ✅ نقر حقيقي يزيد العملات ويستهلك الطاقة، محفوظ في قاعدة بيانات حقيقية (لا يُفقد عند إغلاق التطبيق)
- ✅ استعادة الطاقة تلقائيًا بمرور الوقت حتى وأنت خارج التطبيق
- ✅ الشيفرة اليومية (مكافأة تُطالَب بها مرة كل 24 ساعة)
- ✅ الكومبو اليومي (يُفتح بعد 100 نقرة في اليوم)
- ✅ تغيير المشروع (3 مشاريع بطابع جزائري: حفلة، قهوة الفنك، سوق الحرف)
- ✅ ربط حقيقي ببوت تيليجرام عبر توكن BotFather

## أفكار للتوسّع لاحقًا

- نظام مستويات فعلي (رفع اللاعب من المستوى 1 حسب مجموع العملات)
- متجر ترقيات (زيادة `coins_per_tap` مقابل دفع عملات)
- غرفة الإدارة (CEO Room) — مبنى/موظفين يزيدون الدخل تلقائيًا
- لوحة صدارة عالمية بين اللاعبين (Leaderboard)

هذه غير مُفعّلة حاليًا (الأزرار المقفلة في الشريط السفلي مجرد عرض)، لكن الكود مبني بطريقة يسهل توسيعها منها لاحقًا إذا رغبت.
